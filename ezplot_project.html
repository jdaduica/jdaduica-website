Ezplot: Python Data Visualization Library

A lightweight Python package designed to streamline the creation of common visualizations with customizable and user-friendly functions.

Project Overview

Ezplot is a Python package developed to help data scientists and analysts rapidly generate commonly used plots like line charts, histograms, scatter plots, and heatmaps. It simplifies data exploration by minimizing the amount of code needed to produce effective visualizations.

Library Design

Built core functions to generate customizable plots quickly using matplotlib and seaborn backends.
