Amazon Prime Customer Insights Dashboard

An interactive dashboard to analyze Prime user engagement, churn patterns, and renewal behavior using real-world data.

Project Overview

This dashboard was built to help Amazon understand how users interact with Prime. Using behavioral data and subscription metrics, the tool highlights areas where engagement drops, supporting targeted marketing and retention strategies.

Data Visualization

Developed interactive dashboards using Dash and Plotly.

Data Analysis

Used Pandas and NumPy to analyze engagement, churn rates, and content preferences.

User Interface

Enabled filtering by content type, subscription length, and user demographics.

Technical Implementation

  • Built using Dash (Plotly) for a fully interactive front-end experience.
  • Preprocessed large user activity logs into structured time series data.
  • Implemented multiple filters, dropdowns, and visual highlights to support strategic insights.
  • Hosted locally and optimized for responsive performance.

Data Features

  • User watch history and session duration
  • Renewal status and membership length
  • Genre engagement and peak usage times
